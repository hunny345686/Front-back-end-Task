variables & constants Example

We have 3 ways to declare a variable

var,let and const ,( const we use for constent values like PI )

Var Example:- var variable_name = data (whatever you want to store in this variable in this we have fuction scope with in fuction test(){})

let Example:- let variable_name = data (whatever you want to store in this variable. in let and const we have block scope like within {})

const Example:- const variable_name = data (whatever you want to store in this variable but this value will be constent)
