3 What is the uses of use strict mode in JS?



Not Allowed in Strict Mode

  • Using a variable, without declaring it, is not allowed ex- "use strict"; x = 3.14;
  • Using an object, without declaring it, is not allowed ex- "use strict"; x = {p1:10, p2:20};
  • Deleting a variable (or object) is not allowed. "use strict"; var x = 3.14; delete x;
  • Duplicating a parameter name is not allowed "use strict"; function x(p1, p1) {};
  • Octal numeric literals are not allowed "use strict"; var x = 010;
