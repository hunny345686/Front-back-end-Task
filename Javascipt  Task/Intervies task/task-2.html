01.JS Interview #2 Array Search methods ! Find VS Filter ?



Definition and Usage:-

Find Methos :- The find() method returns the value of the first element in the provided array that satisfies the provided testing function. If no values satisfy the testing function, undefined is returned.

Syntext :- find((element, index, array) => { ... } )

Note:-findIndex() do same like find but return index no of an array

filter() Method : -The filter() method creates a new array with all elements that pass the test implemented by the provided function.

Syntex:- filter((element, index, array) => { ... } )
