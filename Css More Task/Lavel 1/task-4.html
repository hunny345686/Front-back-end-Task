3.how to ways to make a div in center in another div??



There Is 3 ways to make div in center
1. way add css in parent div { display: grid;place-items: center;}
2. way add css in parent div {display: flex;justify-content: center;align-items: center;}
3. way add css in parent div{position: relative;} And in Child Div{position: absolute; bottom: 50%; left: 50%; transform: translate(-50%, -50%);}
