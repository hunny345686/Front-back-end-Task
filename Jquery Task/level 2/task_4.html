jQuery Traversing Siblings , Traversing Sideways in The DOM Tree

There are many useful jQuery methods for traversing sideways in the DOM tree:

  • siblings()
  • nextAll()
  • nextUntil()
  • prev()
  • prevAll()
  • prevUntil()
  • next()
